# Finate - Job Portal Website 🚀

Finate is a modern, fully responsive Job Board and Portal Web UI template. Crafted with clean HTML5, CSS3, and styled using the **Bootstrap 5** framework. 

This project delivers a premium and modern aesthetic for finding jobs, recruiting candidates, reading blogs, and connecting employers with talent.

## 📑 Features
* **Fully Responsive:** Adapts seamlessly to Desktops, Tablets, and Mobile devices.
* **Modern Interface:** Immersive UI with interactive hover scaling, lifting animations, and drop shadows making the cards pop up interactively.
* **100% W3C Validated HTML Code:** Cleanly structured, easily readable, and highly maintainable codebase.
* **Component Rich:** Includes meticulously crafted reusable widgets, job cards, company grids, sidebar navigation, and detailed property views.
* **14 Distinct Pages:** Includes layout for Home, Jobs List, Complete Job Details, Candidate Profiles, Blog Grids, Register/Login flows, 404 Error page, and more!
* **Easy Customization:** Minimal external dependencies with everything properly customized in `style.css`.

## 📁 Project Structure 
- `index.html` — Homepage with categories and brief job previews 
- `jobs.html` / `jobsdetails.html` — Extensible job listings and their detailed views
- `candidates.html` / `candidate-details.html` — Applicant search and profile portals
- `blog*.html` — Blog layouts including details, left sidebar & right sidebar grids
- `about-us.html` — Project overview and team presentation
- `contact.html` — Full working layout for contact requests
- `login.html` / `registration.html` — User authentication portal UI
- `images/` — Contains highly optimized assets, background photos, and company logos
- `style.css` — Core responsive styling and custom interactivity rules

## 💻 Tech Stack
- HTML5
- CSS3 (Custom animations & styling)
- Bootstrap 5 

## 🛠️ Usage / Setup
If you want to view, test, or edit this repository locally:
1. Clone this repository:
   ```bash
   git clone https://github.com/puneetkumawat689-lang/finate-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd finate-website
   ```
3. Open `index.html` inside your web browser. No server environment is strictly required, however using extensions like *VS Code Live Server* provides the best developer experience.
